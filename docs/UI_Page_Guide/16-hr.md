## 16.1 หน้า List พนักงาน

**Route:** `/employees`
**Permission:** `employees.can_view`

- Filter: `branch_id`, `position_id`, `status` (active/inactive)
- Table: ชื่อ, ตำแหน่ง, สาขา, อีเมล, เบอร์โทร, สถานะ

## 16.4 หน้า Attendance (ลงเวลา)

**Route:** `/attendance`
**Permission:** `attendance.can_view`

**มุมมองผู้ดูแล (Admin):**
- ดู attendance ทั้งหมด: `GET /attendance?date=2026-01-15&branch_id=1`
- Filter: วันที่, สาขา, สถานะ
- Sortable: `employee_name` (asc), `date` (asc) — default sort: date asc
- Edit modal → `PUT /attendance/{id}`
- Export CSV: `GET /attendance/export`

**มุมมองพนักงาน (Employee):**
- ปุ่ม "เข้างาน" ขนาดใหญ่ → `POST /attendance/check-in` (ต้องส่ง GPS)
- ปุ่ม "ออกงาน" ขนาดใหญ่ → `POST /attendance/check-out`
- แสดงสถานะวันนี้
- Calendar ประวัติรายเดือน

**Check-in Validation (ระบบตรวจก่อน):**
1. ตรวจ holidays table (date + branch_id)
2. ตรวจ work_schedule_days.is_working
3. ตรวจ employee_schedule_overrides
4. ตรวจ login_start_time ≤ now ≤ login_end_time
5. ตรวจ allowed_ip_range (ถ้ามี)
6. ตรวจ attendance record ซ้ำ
7. คำนวณ late vs grace period (default 15 min)

## 16.5 หน้า วันหยุด (Holidays)

**Route:** `/holidays`
**Permission:** `holidays.can_view`

- Calendar แสดงวันหยุด (จุดแดง)
- Table: วันที่, ชื่อ, สาขา (null=ทุกสาขา)
- Sortable: `date` (asc), `name` (asc) — default sort: date asc
- สร้าง: `POST /holidays { name, date, branch_id }`
  - `branch_id` = null → วันหยุดทุกสาขา
- Query by year: `GET /holidays?year=2026`
